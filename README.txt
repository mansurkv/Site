# Projeto Web - Parte 1

## Tema
Cafeteria Aroma

## Estrutura criada
- /css
- /imagens
- /secreto
- 10 páginas principais em HTML
- robots.txt
- sitemap.xml

## Antes de entregar
1. Troque `SEU-RA-AQUI` pelo seu RA em todas as páginas.
2. Troque `Matheus` pelo seu nome completo, se quiser.
3. Substitua:
   - `GTM-XXXXXXX` pelo seu ID do Google Tag Manager
   - `G-XXXXXXXXXX` pelo seu ID do GA4
   - `COLE_AQUI_O_CODIGO_DO_GSC` pela meta tag do Google Search Console
4. Atualize no `robots.txt` e no `sitemap.xml`:
   - `SEU-USUARIO`
   - `SEU-REPOSITORIO`
5. Publique no GitHub Pages.
6. Adicione `joabefuzaro` como colaborador no repositório.

## Observação
A imagem da Uniube foi deixada como ilustração em SVG para o projeto funcionar sem depender de downloads externos.
Se o professor exigir a marca oficial, substitua `imagens/logo-uniube.svg` pelo arquivo oficial.
